增幅 Amplify 蓝色稀有技能 1 在这个回合，你的下一张能力牌会打出两次。 1 在这个回合，你的下2张能力牌会打出两次。
裂变 Fission 蓝色稀有技能 0 移除所有充能球，每移除一个充能球获得[B]并抽1张牌。消耗。 0 激发所有充能球，每激发一个充能球获得[B]并抽1张牌。消耗。
多重释放 Multi-Cast 蓝色稀有技能 -1 激发你最右侧的充能球X次。 -1 激发你最右侧的充能球X+1次。
彩虹 Rainbow 蓝色稀有技能 2 生成闪电、冰霜、黑暗充能球各一个。消耗。 2 生成闪电、冰霜、黑暗充能球各一个。
重启 Reboot 蓝色稀有技能 0 将你所有未消耗的牌重新洗牌放入抽牌堆。抽4张牌。消耗。 0 将你所有未消耗的牌重新洗牌放入抽牌堆。抽6张牌。消耗。
搜寻 Seek 蓝色稀有技能 0 从抽牌堆中选择1张牌放入你的手牌。消耗。 0 从抽牌堆中选择2张牌放入你的手牌。消耗。
