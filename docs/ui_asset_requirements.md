# UI 美术资源需求清单（中文项目）

## 一、必须先准备（基础版）

### 1) 字体
- 中文主字体（UI 正文）：1 套（Regular/Medium/Bold）
- 中文数值字体（伤害/护甲/能量）：1 套
- 英文/数字补充字体（可选）：1 套

建议格式：`ttf/otf`，需要商业可用授权。

### 2) 卡牌 UI 组件
- 卡牌底板：攻击/技能/能力 各 1 套（含普通、悬浮、拖拽、禁用状态）
- 费用图标：1 套
- 稀有度角标：普通/罕见/稀有
- 卡牌关键字图标：消耗、保留、虚无、X 费等

### 3) 战斗 HUD
- 玩家头像框、敌人头像框
- HP/护甲/能量图标（高清 + 像素版可二选一）
- 意图图标：攻击、防御、强化、减益、多段攻击、未知
- 状态图标（至少 20 个高频）：力量、敏捷、虚弱、易伤、中毒、金属化、荆棘等

### 4) 地图与节点
- 节点图标：普通战/精英/休息/事件/商店/Boss
- 连线样式 2 套（默认/高亮）
- 地图背景 2 张（Act1 基础版）

### 5) 反馈与特效
- 打击、格挡、治疗、抽牌、弃牌、消耗、中毒跳字
- 关键 UI 动效切片：按钮 hover、面板弹入、奖励揭示

## 二、第二阶段补齐（扩展版）

- 角色立绘（4 角色）
- 敌人立绘分层（普通/精英/Boss）
- 事件插图（先 15 张）
- 遗物插图（先 50 张）
- 药水插图（先 30 张）
- 商店 NPC 与场景图

## 三、导出规范（避免返工）

- 图标：`64x64` 或 `128x128`，透明 PNG
- 卡图：建议 `512x768`（便于后续缩放）
- 背景图：`1920x1080` 基础分辨率
- 命名：`模块_类型_名称_状态.png`
- 目录：`assets/ui`, `assets/cards`, `assets/icons`, `assets/vfx`, `assets/map`

## 四、你需要给我的美术输入

1. 美术风格方向：像素/手绘/UI 扁平化（三选一或混合）
2. 颜色基调：明亮/暗色/中性
3. 是否保留“原版辨识度”还是“明显差异化”
4. 第一批外包范围：只做 UI 皮肤，还是连角色敌人一并做

确认这 4 项后，我可以把资源清单进一步拆成“可直接发包的任务单”。
