# 埋骨钱（敌人占位）提示词

角色硬特征（不要丢）：
- 男性，偏贵公子/诡谲谋士气质，神态从容但危险
- 金色长发，发量充足，带发冠/发饰，面部干净俊朗
- 武器为金绿配色的弧形刃（短镰/弯刃感），不是直剑
- 服装为黑金主调长袍，带流动披摆，点缀翡翠绿色饰物与钱纹/古典纹样
- 气质要“富贵 + 阴冷算计”，不是骷髅怪物，不是纯正派英雄

可直接复制（正向提示词）：
以我上传的两张“埋骨钱参考图”（立绘图+设计图）为共同角色依据，生成同一角色的统一立绘。角色为金发男性，贵气且带危险感，长发与头饰清晰；服装以黑金为主，加入翡翠绿饰件与古典纹样，披摆和丝带有流动感。主武器必须是金绿弧形刃类武器（非直剑），与角色手势形成精致但致命的观感。人物姿态改为全身站姿（不要复用参考图动作），站姿从容、带轻蔑感，强调“优雅与威胁并存”。整体画风严格采用“杀戮尖塔式”手绘卡牌风：形体概括明确、明暗对比强、可读性优先、带少量墨迹与金色碎屑笔触。输出 1024x1024，透明背景 PNG，只保留角色与少量黑墨/金色粒子特效，不要完整场景。

可直接复制（负面提示词）：
骷髅脸、腐尸化、怪物化、重口血腥、圣骑士正派风、朴素村民装、白银重甲、现代西装、枪械、直剑主武器、Q版、幼态、女性化、荧光霓虹、紫色主调、复杂背景、建筑地景、文字水印、logo、低清、肢体错误、面部崩坏。

参数建议：
- 分辨率：1024x1024
- 背景：透明（alpha）
- 构图：单人全身，角色占画面 65%~80%
- 风格强度：中高（保证角色身份识别优先）
