# 离恨烟（敌人占位）提示词

角色硬特征（不要丢）：
- 女性，身形修长轻盈，气质冷静、危险、带一点疏离感
- 深色长发，双髻/发团结构，红色发珠与粉红飘带明显
- 核心武器是大尺寸伞形法器（黑白金结构），可悬浮在身后或侧后方
- 服装为青白冷色系，带墨色层次与金属小饰件，腿部线条轻巧
- 伴随墨烟/水墨泼洒感特效，颜色以黑、青、白、淡粉为主

可直接复制（正向提示词）：
以我上传的两张“离恨烟参考图”（立绘图+设计图）为共同角色依据，生成同一角色的统一立绘。角色为冷艳轻灵的女性，深色长发、双髻发饰、红色小发珠与粉色长飘带；武器必须是大型伞形法器（黑白金主结构，带古典纹样），伞体可悬浮于角色后上方，形成强识别剪影。服装为青白与墨色过渡的东方感轻装，辅以少量金属装饰与丝带，身形纤细但不弱。构图改为全身站姿（不要复用参考图动作），站姿优雅、重心轻、手部与伞的位置关系清晰。整体画风严格采用“杀戮尖塔式”手绘卡牌风：轮廓清楚、明暗分块、笔触感与水墨动势并存，强调卡面识别度。输出 1024x1024，透明背景 PNG，仅保留角色与少量墨烟/丝带特效，不要完整场景。

可直接复制（负面提示词）：
重甲战士风、男性体型、Q版、幼态、甜美偶像风、现代雨伞、科技伞、枪械、巨大翅膀、过度裸露、荧光高饱和、紫色主调、背景建筑、地面场景、文字水印、logo、糊脸、手指错误、肢体缺失、伞体结构模糊。

参数建议：
- 分辨率：1024x1024
- 背景：透明（alpha）
- 构图：单人全身，角色占画面 65%~80%
- 风格强度：中高（水墨动势可见，但角色结构优先）
