# 角色立绘提示词（统一风格）

用途：
- 为 `霜北刀`（主控）/`离恨烟`（敌人占位）/`埋骨钱`（敌人占位）生成统一风格立绘。
- 每次生成时都要同时挂两张参考图：`立绘参考图 + 设计参考图`。

统一硬性要求：
- 分辨率：`1024x1024`
- 输出：`PNG 透明底（alpha）`
- 构图：`全身站姿`
- 画风：`明确对齐杀戮尖塔风格（2D 手绘、剪影清晰、可读性优先）`
- 禁止：直接复刻参考图姿势

文件说明：
- `00_通用生成规范.md`：通用规则与参数
- `01_统一风格母提示词.md`：三角色共用风格母提示词
- `10_霜北刀_主控提示词.md`
- `11_离恨烟_敌人占位提示词.md`
- `12_埋骨钱_敌人占位提示词.md`
- `20_角色提示词套件_v2.md`：可直接投喂 AI 的整套提示词（含批量生成与二次修正）
