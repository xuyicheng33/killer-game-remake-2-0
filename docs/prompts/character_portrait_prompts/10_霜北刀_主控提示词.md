# 霜北刀（主控）提示词

角色硬特征（不要丢）：
- 男性，北境重装武将气质，身材高大结实，肩背宽，压迫感强
- 黑白双色长发，高马尾，额前有冷色发束，面容冷峻
- 不对称重甲：一侧有大型兽首护肩/臂盾（金属金+青绿点缀），另一侧更轻便
- 主武器必须是超大霜蓝晶体戟刃长兵（不是普通剑/刀），有冰裂纹与寒光
- 黑灰主服装，少量红色腰带飘带，辅以暗金纹样与青绿色饰件

可直接复制（正向提示词）：
以我上传的两张“霜北刀参考图”（立绘图+设计图）为共同角色依据，生成同一角色的统一立绘。角色为男性重装战士，黑白双色长发高束，神情冷硬，身形强壮，不对称甲胄结构明显，单侧大型兽首护肩与臂甲突出；手持超大霜蓝晶体戟刃长兵，武器体量大、冰晶切面清晰、寒霜能量可见。服装主色黑灰，辅色冰蓝、暗金、青绿，红色腰带飘带作点缀。构图为全身站姿（非参考图原姿势），三分之四视角，武器斜向落地或侧持，剪影清晰、重心稳定、卡牌内可读性强。整体画风严格为“杀戮尖塔式”手绘卡牌风：高识别轮廓、块面光影、略带笔触与墨迹动势，不做二次元厚涂脸，不做写实照片感。输出 1024x1024，透明背景 PNG，只保留角色与少量寒霜/墨迹特效，不要场景。

可直接复制（负面提示词）：
普通直剑、细剑、现代武器、枪械、机甲、科幻盔甲、轻飘美少年体型、Q版、幼态、女性化、表情温和微笑、武器过小、双手空手、护肩消失、红色占比过高、暖色主调、复杂背景、完整地面场景、文字水印、logo、边框、低清晰度、糊脸、断手断脚、额外肢体、错误透视。

参数建议：
- 分辨率：1024x1024
- 背景：透明（alpha）
- 构图：单人全身，角色占画面 70%~85%
- 风格强度：中高（优先保证角色识别，不要过度抽象）
