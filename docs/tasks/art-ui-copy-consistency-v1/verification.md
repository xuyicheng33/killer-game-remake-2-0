## 术语对照表（改前 -> 改后）

- `生命 80/80   金币 99` -> `生命：80/80   金币：99`
- `选择其一：休息（回血）或升级（强化牌组第1张卡）` -> `选择一项：休息（恢复生命）或升级（强化牌组第 1 张卡）`
- `休息（恢复 20% 生命，最低 6）` -> `休息（恢复 20% 生命，至少 6）`
- `战后奖励` -> `战斗奖励`
- `跳过（仅拿金币并返回地图）` -> `跳过奖励（仅领取金币）`
- `无卡牌奖励（可直接返回地图）` -> `当前无卡牌奖励，可直接继续前进。`
- `买卡` -> `购买卡牌`
- `删卡` -> `移除卡牌`
- `金币不足，无法删卡。` -> `金币不足，无法移除卡牌。`
- `删卡失败。` -> `移除卡牌失败。`
- `离开商店并继续` -> `继续前进`
- `完成操作后可离开商店。` -> `完成操作后可继续前进。`
- `抽:0 手牌:0 弃:0 消:0` -> `抽牌堆：0  手牌：0 / 弃牌堆：0  消耗堆：0`（双行）
- `本轮结束` -> `结束本次远征`
- `战后奖励：%s` -> `战斗奖励：%s`
