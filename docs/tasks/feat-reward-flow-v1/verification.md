### 用例 1：胜利后进入奖励页并获得金币

- 前置：运行入口场景 `scenes/app/app.tscn`
- 步骤：
  1. 在地图选择任意 `BATTLE/ELITE` 节点进入战斗。
  2. 打赢战斗，出现胜利面板后点击“Continue”。
  3. 确认进入“战后奖励”页面，页面显示 `金币：+<x>`。
  4. 点击“跳过（仅拿金币并返回地图）”，回到地图。
  5. 观察地图页头部金币数已增加（与奖励页显示一致）。
- 期望：
  - 战斗胜利后进入奖励页（不是直接回地图）。
  - 奖励页展示金币数，确认后 `RunState.gold` 增加并回到地图。

### 用例 2：三选一卡并写回 RunState.deck

- 前置：运行入口场景 `scenes/app/app.tscn`
- 步骤：
  1. 在地图选择任意 `BATTLE/ELITE` 节点进入战斗并获胜，点击“Continue”进入奖励页。
  2. 在奖励页点击任意 1 张卡选项（按钮文本包含 `warrior_*` 卡 id）。
  3. 回到地图后，进入下一场战斗（再选一个 `BATTLE/ELITE` 节点）。
  4. 在玩家回合观察手牌/抽牌堆是否可能抽到刚才选择的卡（允许重复，概率相关）。
- 期望：
  - 奖励页展示 3 个可选卡项。
  - 选择其中 1 张后，该卡加入本局牌组（当前实现为 `run_state.player_stats.deck`），并返回地图。
