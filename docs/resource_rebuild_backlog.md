# 资源替换重构任务清单（美术 + 音乐 + 音效）

## 目标

在不改变玩法逻辑的前提下，分批替换教程项目资源，形成可商用、可持续迭代的资源体系。

## 美术替换包

### 包 A：UI 与图标（优先）
- 字体：中文主字体 + 数值字体
- 卡牌底板：攻击/技能/能力 + 稀有度边框
- 战斗HUD：HP/格挡/能量/状态/意图图标
- 地图节点：普通/精英/休息/事件/商店/Boss
- 输出：`assets/ui/**`

### 包 B：角色与敌人
- 角色立绘（第一批至少1个）
- 敌人立绘（Act1 普通 + 精英 + Boss）
- 战斗特效图集（受击、格挡、中毒、强化）
- 输出：`assets/units/**`, `assets/vfx/**`

### 包 C：事件与场景
- 事件插图（首批10个）
- 地图背景、营火背景、商店背景
- 奖励页与结算页背景
- 输出：`assets/backgrounds/**`, `assets/events/**`

## 音乐与音效替换包

### 包 M1：BGM
- 主菜单BGM
- 地图BGM（Act1）
- 战斗BGM（普通/精英/Boss）
- 胜利/失败短音乐

### 包 M2：SFX
- 出牌、抽牌、弃牌、消耗
- 受击、格挡、治疗、状态触发
- UI按钮、页面切换、奖励揭示

### 包 M3：混音与总线
- `Master/BGM/SFX/UI/Ambience` 五总线
- 默认音量与限制峰值
- 动态ducking（可选）

## 每个资源包任务验收

1. 替换后无缺图（无粉色贴图）
2. 替换后无缺音（无静默触发）
3. 1080p 与 16:9 下布局不溢出
4. 资源版权与授权信息可追溯（保留在 `docs/assets_licenses.md`）
